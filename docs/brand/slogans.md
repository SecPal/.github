# Slogans

Slogans are short brand lines, not sentences. They do not end with a period.

## Approved Slogans

| Brand                | Exact slogan                         |
| -------------------- | ------------------------------------ |
| SecPal               | `A guard's best friend`              |
| GuardGuide by SecPal | `Clear instructions for every shift` |

## Usage Rules

- Use the exact capitalization and punctuation shown above.
- Do not add a period, exclamation mark, or trailing full stop.
- Do not rewrite the slogan for local repository needs.
- Do not combine the slogan with technical license identifiers or package metadata.
- Use slogans in public-facing surfaces where a short brand line is expected, such as profile text, website hero support copy, app metadata, and product footers.

## Rationale

Keeping slogans period-free makes them work as labels, lockups, and footer fragments. It also avoids inconsistent punctuation between websites, app surfaces, and repository profile copy.
